# Kadane
Kadane algorithm finds the maximum sum of contiguous elements inside an array.

## Complexity
With $n$ the number of elements inside the input array:
$$
O(n)
$$

## Implementations
### Python
```py
# implementations/kadane.py
```

### Rust
```rust
// implementations/kadane.rs
```

##### Input
```
[1, 2, -5, 10, 8]
```

##### Output
```
18
```
